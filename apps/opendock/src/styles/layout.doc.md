# layout.css

Layout styles for the app shell — titlebar, sidebar, content area, and auth page.

## Sections
- **App Shell** — top-level flex column layout
- **Titlebar** — custom window chrome with drag region
- **Sidebar** — collapsible navigation sidebar
- **Content Area** — main scrollable area
- **Auth Layout** — centered card for login/register

## Used by
- Imported in `main.tsx` after `base.css`
