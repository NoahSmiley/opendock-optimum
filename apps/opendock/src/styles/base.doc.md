# base.css

Global CSS reset, custom properties (colors, sizing, typography, spacing), and base element styles.

## Sections
- **Reset** — box-sizing, margin/padding normalization
- **Custom Properties** — all design tokens for the app
- **Scrollbar** — custom dark scrollbar styling
- **Element Resets** — button, input, link defaults

## Used by
- Imported in `main.tsx` as the first stylesheet
