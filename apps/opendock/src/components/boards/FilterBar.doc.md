# FilterBar

Dropdown-based filter controls for the kanban board. Filters by priority, assignee, and label.

## Props
- `labels`, `members` — Options for dropdowns
- `priority`, `assignee`, `label` — Current filter values
- `onPriorityChange`, `onAssigneeChange`, `onLabelChange` — Callbacks

## Used by
- `BoardsPage.tsx`
