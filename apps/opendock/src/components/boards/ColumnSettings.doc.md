# ColumnSettings

Settings panel for listing, creating, and deleting board columns.

## Props
- `board` — The Board whose columns are managed

## Used by
- `BoardSettingsModal.tsx`

## Dependencies
- `@/stores/boards/actions` (createColumn, deleteColumn)
- `lucide-react` (Plus, Trash2, GripVertical)
- `@/stores/boards/types` (Board)
