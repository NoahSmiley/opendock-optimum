# BoardSettingsModal

Modal dialog for managing board settings with tabs for columns, labels, and board info.

## Props
- `board` — The Board object to configure
- `onClose` — Callback to close the modal

## Used by
- `BoardsPage.tsx`

## Dependencies
- `ColumnSettings`, `LabelSettings`
- `lucide-react` (X)
- `@/stores/boards/types` (Board)
