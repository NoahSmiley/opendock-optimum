# ticket.rs

Ticket-related request DTOs covering CRUD, reordering, comments, and sprints.

## Exports
- `CreateTicketReq` — create ticket request body
- `UpdateTicketReq` — update ticket request body
- `ReorderReq` — ticket reorder/move request
- `CommentReq` — add comment request body
- `CreateSprintReq` — create sprint request body
